# About

Gym Diary is a lightweight personal workout planner built in Python to work as a quick note. It lets you record the exercises you perform on each day of the week, and every time you save your entries, the exact date and time of the save is automatically recorded alongside them.

## Features

-  Organize exercises by day of the week
-  Multi-language support with an interactive toggle button
-  Automatic timestamp on every save
-  Persistent storage — your diary is saved to a file
-  Pure Python — no external dependencies required

1. Run the program:
   ```bash
   python main.py
   ```

## How It Works

1. Choose a day of the week
2. Choose your muscular group
3. Enter the exercises you performed
4. Save your entry — the program automatically records the exact time of saving
5. Your workout history is stored and ready to be reviewed anytime
